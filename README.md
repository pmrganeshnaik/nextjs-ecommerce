# Next.js E-commerce Store

This is a basic e-commerce store built with Next.js 14, Tailwind CSS, and Supabase.

## Getting Started

1.  Clone the repository:

    ```bash
    git clone <repository-url>
    cd nextjs-ecommerce
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

3.  Set up your `.env.local` file with your Supabase credentials.  See `.env.local.example` for the required variables.

4.  Run the development server:

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

Deploy this Next.js app to Vercel or any other platform that supports Next.js.

Ensure you have set up your environment variables (Supabase URL and Anon Key) in your deployment environment.
